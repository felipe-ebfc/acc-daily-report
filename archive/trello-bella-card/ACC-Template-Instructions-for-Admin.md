**TEMPLATE NAME:** Daily Report - All Trades

**WHERE TO CREATE IT:**

1. Go to ACC > Folsom Care Complex at Broadstone Phase II
2. Click "Forms" in the left sidebar
3. Click "Templates" tab at the top
4. Click "+ Create Template" button
5. Select "Start from scratch"
6. Name it: Daily Report - All Trades

**SECTIONS TO ADD:**

Section 1: Work Log (Table)
- Set as "Table" field type
- Columns: Company, Discipline, Role, # Workers, # Regular Hours, # OT Hours, Work Performed
- Pre-populate Company column with all trades below

Section 2: Materials
- Table with columns: Material, Quantity, Unit, Comment

Section 3: Equipment
- Table with columns: Equipment, Quantity, Hours Used, Comment

Section 4: Signature
- Signature field for Foreman
